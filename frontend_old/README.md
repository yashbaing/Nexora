# Stockwave Frontend

React frontend for the Stockwave trading application.

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Build

```bash
npm run build
```

## Features

- User authentication (login/register)
- Real-time stock prices via WebSocket
- Portfolio management
- Multiple payment methods (UPI, Crypto, Card)
- Order history
- Settings and preferences
- Responsive design

## Pages

1. Login - User authentication
2. Register - Account creation
3. Home - Stock market overview
4. Stock Detail - Detailed stock information with charts
5. Portfolio - User holdings
6. Wallet - Fund management
7. Deposit - Multiple payment methods
8. Order History - Trading history
9. Settings - User preferences
